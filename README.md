# Public Domain Aeronautical Software
List of Public domain aeronautical software maintained by pdas www.pdas.com and some other intresting projects

<ol>
<li>

<li>

<li>

</li>

Non-standard atmosphere routines (hot, cold, polar, tropical) are also included. <strong>&nbsp;/&nbsp;ATMOS</strong></div>

<li>
